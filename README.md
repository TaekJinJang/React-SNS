# React-SNS
 
## Teck Stack
- FrontEnd
  - Next.js (SSR)
  - React Hooks
  - Redux / Redux Saga
  - Ant Design
  - Styled Components

- BackEnd
  - Node.js
  - Express
  - MySQL (Sequelize)
  - Bcrypt

- production
  - AWS
  - EC2 / S3
  - Route 53
  - nginx
